TASK 4 – Spam SMS Detection

This project is a machine learning model developed using Python to classify SMS messages as Spam or Legitimate (Ham). It is created as part of my CODSOFT Machine Learning Internship – Task 4. The model uses Natural Language Processing (NLP) techniques to analyze SMS messages and identify whether they are spam.

📌 Project Description

The Spam SMS Detection System automatically identifies unwanted spam messages using machine learning. The model processes SMS text data, converts it into numerical features using TF-IDF (Term Frequency–Inverse Document Frequency), and applies classification algorithms to detect spam messages.

The system learns patterns from labeled SMS datasets and predicts whether a new SMS message is Spam or Not Spam (Ham).

Text Processing: Cleans and prepares SMS messages for analysis.
Feature Extraction: Uses TF-IDF to convert text into numerical vectors.
Classification: Machine learning algorithms classify messages as spam or legitimate.

This project helps beginners understand text preprocessing, NLP techniques, and machine learning classification.

🚀 Features

Classifies SMS messages as Spam or Legitimate

Uses Natural Language Processing (NLP)

Converts text into numerical format using TF-IDF Vectorization

Applies Machine Learning Classification Algorithms

Predicts spam messages with trained models
